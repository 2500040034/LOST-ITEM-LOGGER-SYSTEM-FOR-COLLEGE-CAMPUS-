#  Lost Item Logger System for College Campus (LIL-CAMP)

##  Overview
The Lost Item Logger System for College Campus (LIL-CAMP) is a simple digital solution designed to manage lost and found items efficiently within a college environment. 

In large campuses, students and staff frequently lose personal belongings such as ID cards, books, wallets, and electronic gadgets. Traditional methods like notice boards or manual registers are often unorganized and time-consuming. This system replaces those methods with a structured and centralized approach.

## Objective
The primary objective of this project is to create an easy-to-use system that helps users:
- Report lost items  
- Record found items  
- Search and match item details  
- Manage all records in one place  

## Key Features
-  Add details of lost items

## Technology Stack
- Programming Language: Python  
- Concept Used: File Handling  

## Benefits
- Reduces manual work  
- Saves time in searching items  
- Improves organization  
- Easy to use and understand  
- Increases chances of recovering lost items  

## Conclusion
LIL-CAMP provides a reliable and efficient way to manage lost and found items in a college campus. By digitizing the process, it ensures better tracking, faster access, and improved coordination between users.
